# DepositOS

DepositOS is a personal operating system for tracking the daily deposits that build health, faith, family, business, and self.

## What it includes

- configurable deposit definitions with pillars, schedules, targets, and units
- previous/next-day navigation plus a full calendar and daily ledger
- editable historical entries, numeric measurements, completion state, and notes
- 30-day pattern insights calculated from recorded data
- saved Cope Differently reflections and Sunday Reset reviews
- private durable storage through Vercel Blob with optimistic concurrency protection
- responsive desktop and mobile interfaces

## Development

```bash
npm install
npm run build
npm test
```

Production deployments are built automatically from `main` by Vercel.
